# Cultural Tourism in India

A responsive 3-page static website showcasing India's cultural heritage with an intelligent chatbot guide.

---

## Features

- **3 Pages**: Home, About, Contact
- **6 Destinations**: Taj Mahal, Jaipur, Varanasi, Hampi, Kerala, Delhi
- **4 Festivals**: Diwali, Holi, Durga Puja, Pongal
- **Smart Chatbot**: 30+ responses about destinations, festivals, and travel tips
- **Fully Responsive**: Works on mobile, tablet, and desktop

---

- CSS3
